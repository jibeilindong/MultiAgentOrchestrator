# OpenClaw 网关远程 Agent 执行指南

## 概述

本指南介绍如何使用 MultiAgentOrchestrator 通过 OpenClaw 网关远程执行 Agent 任务。

## 快速开始

### 1. 配置连接

在应用设置中配置 OpenClaw 网关连接：

- **Host**: OpenClaw 网关地址（默认：`127.0.0.1`）
- **Port**: 网关端口（默认：`18789`）
- **Timeout**: 执行超时时间（秒）

### 2. 创建工作流

1. 在工作流编辑器中创建新的工作流
2. 添加 Agent 节点
3. 配置节点连接到关系

### 3. 执行工作流

1. 选择要执行的工作流
2. 点击 "Execute" 按钮
3. 观察执行进度和日志

## 功能说明

### 执行状态显示

- **进度条**: 显示当前执行步骤和总步骤数
- **状态指示器**: 显示 OpenClaw 网关连接状态
  - 🟢 绿色: 已连接
  - 🟡 黄色: 连接中
  - 🔴 红色: 连接错误
  - ⚪ 灰色: 已断开

### 实时日志

执行过程中可以查看实时日志：
- 点击 "Logs" 按钮显示日志面板
- 日志级别: INFO, WARN, ERROR, SUCCESS
- 自动滚动到最新日志

### 错误处理

- **超时处理**: 执行超过设定时间自动终止
- **连接断开**: 自动检测连接状态，提示用户
- **失败恢复**: 节点执行失败会记录日志并继续执行

### 回滚机制

- **暂停执行**: 点击暂停按钮保存当前状态
- **恢复执行**: 重新连接后可恢复执行
- **回滚检查点**: 可回滚到上一个成功节点

## 常见问题

### Q: 连接失败怎么办？
1. 检查 OpenClaw 网关是否运行
2. 确认 Host 和 Port 配置正确
3. 尝试点击 "Test Connection" 测试

### Q: 执行超时怎么办？
1. 增加超时时间设置
2. 检查网络连接
3. 查看错误日志

### Q: 如何查看执行历史？
- 执行结果保存在 "Results" 面板中
- 可查看每个节点的执行状态和输出
