# Multi-Agent-Flow Workflow Editor Guide

工作流编辑器是 Multi-Agent-Flow 的核心功能，用于设计和管理多智能体工作流。

## 概述

工作流编辑器支持三种视图模式：
- **List View** (列表视图) - 以列表形式显示所有智能体
- **Grid View** (网格视图) - 以卡片网格形式显示智能体
- **Architecture View** (架构视图) - 以可视化架构图形式显示智能体和连接

## 界面组件

### 工具栏

工具栏位于编辑器顶部，包含以下功能：

| 组件 | 功能 | 说明 |
|------|------|------|
| 视图选择器 | 切换三种视图模式 | List / Grid / Architecture |
| Connect 按钮 | 连接模式 | 点击后选择源智能体和目标智能体创建连接 |
| Test 按钮 | 测试执行 | 运行工作流测试 |
| 保存状态 | 自动保存指示 | 显示自动保存状态 |
| Save 按钮 | 手动保存 | 手动保存项目到文件 |

### 右键菜单

在智能体节点上右键点击显示操作菜单：

| 操作 | 说明 |
|------|------|
| Open | 在属性面板中打开智能体 |
| Copy | 复制智能体到剪贴板 |
| Paste | 粘贴剪贴板中的智能体 |
| Export | 导出智能体为 JSON 文件 |
| Edit SOUL.md | 编辑智能体的 SOUL.md 配置 |
| Manage Skills | 管理智能体技能 |
| Configure Permissions | 配置智能体权限 |
| Duplicate | 复制智能体创建副本 |
| Reset | 重置智能体状态 |
| Delete | 删除智能体 |

### Toast 反馈

执行操作后会显示 Toast 消息提示：
- ✅ 成功 - 绿色提示
- ❌ 错误 - 红色提示  
- ℹ️ 信息 - 蓝色提示

## 使用指南

### 创建智能体连接

1. 点击工具栏中的 **Connect** 按钮
2. 在视图中点击源智能体（连接起点）
3. 点击目标智能体（连接终点）
4. 连接自动创建

### 切换视图

点击工具栏中的视图按钮切换：
- 📋 List - 列表视图
- ⊞ Grid - 网格视图
- 🔗 Architecture - 架构视图

### 测试工作流

1. 点击工具栏中的 **Test** 按钮
2. 系统按拓扑顺序执行智能体
3. 底部显示执行进度面板

### 数据持久化

**自动保存**
- 每 60 秒自动保存到 `~/Library/Application Support/Multi-Agent-Flow/AutoSave/`
- 工具栏显示自动保存状态

**手动保存**
- 点击工具栏中的 **Save** 按钮
- 选择保存位置和文件名（.maoproj 格式）

**加载项目**
- 使用 File > Open 加载 .maoproj 文件

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| ⌘S | 保存项目 |
| ⌘O | 打开项目 |
| ⌘N | 新建项目 |

## 数据流

```
Workflow → Nodes (智能体) → Edges (连接) → Execution Order
```

工作流由节点（智能体）和边（连接）组成。执行时按拓扑顺序依次调用各智能体。

## 故障排查

### 连接创建失败
- 确保不是自连接（同一智能体）
- 检查智能体是否存在

### 保存失败
- 检查文件写入权限
- 确保磁盘空间充足

### 测试执行无响应
- 检查智能体配置
- 查看日志面板获取详细信息
